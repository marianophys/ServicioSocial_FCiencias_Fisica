# ServicioSocial_Fisica_FCiencias
Aplicación de Métodos de Machine Learning para la Clasificación de Eventos en Colisiones de Partículas usando Datos Abiertos del CERN


## Datasets

Los archivos `.root` de ATLAS no se incluyen en este repositorio
debido a restricciones de tamaño de GitHub.

Descargar los datos desde:
[ATLAS Open Data 13 TeV](https://opendata.cern/record/15007)]

Con las claves:
* mc_345324.ggH125_WW2lep.exactly2lep.root
* mc_363492.llvv.exactly2lep.root'
* mc_410011.single_top_tchan.exactly2lep.root
* mc_361108.Ztautau.exactly2lep.root
